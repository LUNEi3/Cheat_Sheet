  • Header: In this section, is used to represent the introductory content or navigational links of a section or page.

  • nav: Nav is used to represent a section of the page intended for navigation. It typically contains a list of links to other pages or sections within the current page.

  • Main: In this section, is used to represent the dominant content of the Webpage which contains the contents that relate to the central ideas/topic of it.

  • section: Section is a container for different content related to the main topic that separates into each section.
    Ex. figure, figcaption, aside, article, blockquote, or any input tags.

  • Footer: In this section, is used to represent the footer of a section or page, typically containing information about the author, copyright information, or links to related documents.
